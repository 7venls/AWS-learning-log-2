【 your-name-s3-static-xxxx 】

【 全部 ON 】

# 5.バケットポリシーを設定（公開許可）

### 5-3. 編集

### 5-4. 以下を貼り付け（バケット名は自分のに変更）

{

 "Version": "2012-10-17",
 "Statement": [
 {
  "Effect": "Allow",
  "Principal": "*",
  "Action": "s3:GetObject",
  "Resource": "arn:aws:s3:::naoya-s3-static-site-2025 /*"
  }
 ]
}


注意：自分の実際のバケット名にしないと不明なエラーが出る。

# 8. 失敗したときのチェック表

【 403 AccessDenied 】

・ブロックパブリックアクセス解除した？

・バケットポリシーのバケット名合ってる？

・GetObject になってる？

【 404 Not Found 】

・ファイル名 index.html？

・静的ホスティング有効？
