# S3 で静的サイト公開

# 1.事前準備

## ローカルPCでやること

### ①メモ帳 / VS Code を開く

### ②以下をそのまま貼る

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>S3 Static Site</title>
</head>
<body>
  <h1>Hello S3</h1>
  <p>Week3 Day4</p>
</body>
</html>

### ③ファイル名を index.html にして保存

・保存場所はどこでもOK（デスクトップ推奨）

# 2.S3 バケット作成

### 2-1. AWS コンソール ⇒ S3

### 2-2. バケットを作成

### 2-3. 一般的な設定(AWS リージョン)

【 アジアパシフィック (東京) ap-northeast-1 】

・リージョン：今まで使ってるリージョン

### 2-4. 一般的な設定(バケットタイプ)

【 凡用 】

### 2-5. 一般的な設定(バケット名)

【 your-name-s3-static-xxxx 】

### 2-6. オブジェクト所有権

【 ACL 無効 (推奨) 】

### 2-7. このバケットのブロックパブリックアクセス設定

【 全部 ON 】

### 2-8. デフォルトの暗号化(暗号化タイプ)

【 SSE-S3 】
