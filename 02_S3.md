# オブジェクト操作（PUT / GET）

# ①PUT（オブジェクトをアップロード）

## 1.S3 ⇒ 作成済みバケットをクリック

例 : my-study-s3-naoya-001

## 2.アップロードボタンをクリック

## 3.ファイルを用意する（ローカルPC）

【 Windowsの場合メモ帳で作るのが一番簡単 】

①「メモ帳」を開く。

②【 S3 PUT test 】中身にこれを書く。

③メニュー「ファイル」⇒「名前を付けて保存」

保存時の注意（超重要）

・保存場所：デスクトップ

・ファイル名：test.txt

## 4.ファイルを追加 ⇒ test.txt（さっき作ったもの）を選択。

## 5.アップロード

・成功すると test.txt がオブジェクト一覧に表示される。

・これが PUT。

・PUT = ローカル → S3 に保存

・CloudTrailでは

👉 PutObject API として記録される
