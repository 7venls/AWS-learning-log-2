# オブジェクト操作（PUT / GET）

# ①PUT（オブジェクトをアップロード）

## 1.S3 ⇒ 作成済みバケットをクリック

例 : my-study-s3-naoya-001

## 2.アップロードボタンをクリック

## 3.ファイルを用意する（ローカルPC）

【 Windowsの場合メモ帳で作るのが一番簡単 】

①「メモ帳」を開く。

②【 S3 PUT test 】中身にこれを書く。

③メニュー「ファイル」⇒「名前を付けて保存」

保存時の注意（超重要）

・保存場所：デスクトップ

・ファイル名：test.txt

## 4.ファイルを追加 ⇒ test.txt（さっき作ったもの）を選択。

## 5.アップロード

・成功すると test.txt がオブジェクト一覧に表示される。

・これが PUT。

・PUT = ローカル → S3 に保存

・CloudTrailでは

👉 PutObject API として記録される

# ②GET（オブジェクトをダウンロード）

## 1.オブジェクトを選択 ⇒ test.txt をクリック。

## 2.ダウンロード

・右上「ダウンロード」をクリック。

## 3.確認

・ローカルPCに保存される。

・中身を開いて確認。

【 S3 PUT test 】これがGET。

# ③オブジェクトURLで GET を試す（403体験）

## 1.オブジェクトURLをコピー

・test.txt の詳細画面。

・オブジェクトURL をコピー。

例 : https://my-study-s3-naoya-001.s3.ap-northeast-1.amazonaws.com/test.txt

## 2.ブラウザでアクセス

・新しいタブでURLを貼り付け。

## 3.結果

❌ 403 AccessDenied

【 なぜ 403 になるのか？ 】

・パブリックアクセスをブロックしている。

・バケットポリシーで匿名GETを許可していない。

・ACLは無効。

「PUT / GET できる = 自分だけ」
